# HTML5扫描枪录入工具 #

自动识别来自扫描枪录入的条码并进行提交，屏蔽来自键盘的录入。

# 版本号命名规则 #

X: 版本号, Y: 修订号, Z: 迭代次数

> 如：Ver.X.Y.Z 代表第X版，第Y次修订，第Z次迭代。
